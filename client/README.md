# Infographic App Client

Клиентская часть веб-приложения для генерации инфографики по учебной статистике.

## Назначение

Flutter Web-клиент предоставляет пользовательский интерфейс для авторизации, регистрации, работы с учебными данными, генерации инфографики, сохранения результатов, экспорта и административного управления.

## Используемые технологии

- Flutter Web;
- Dart;
- BLoC;
- go_router;
- fl_chart;
- http / dio;
- shared_preferences;
- file_picker;
- csv.

## Архитектура клиентской части

Проект организован по feature-first структуре. Основные функциональные области вынесены в отдельные модули.

```text
lib/src/
├── app/                     # Конфигурация приложения и роутинг
├── core/                    # Общие утилиты, API-клиент, темы, ошибки
├── features/
│   ├── auth/                # Авторизация и регистрация
│   ├── admin/               # Административная панель
│   ├── dashboard/           # Главная страница
│   ├── educational_data/    # Учебные данные
│   ├── infographic_builder/ # Генератор инфографики
│   ├── saved_infographics/  # Сохранённые инфографики
│   └── profile/             # Профиль пользователя
└── main.dart
```

Клиентская часть использует MWWM-подход: пользовательский интерфейс, состояние экрана и работа с данными разделены по слоям. Для управления состоянием и пользовательскими событиями применяется BLoC.

## Основные пользовательские сценарии

### Авторизация

Пользователь вводит email и пароль. После успешной авторизации сервер возвращает JWT-токен, который сохраняется на клиенте.

### Работа с учебными данными

Пользователь может просматривать, добавлять, изменять и удалять:

- учебные группы;
- студентов;
- дисциплины;
- учебные периоды;
- оценки;
- посещаемость.

### Импорт CSV

В разделе учебных данных доступен импорт CSV для:

- студентов;
- оценок;
- посещаемости.

Файл должен быть сохранён в UTF-8.

### Генерация инфографики

Пользователь выбирает данные, тип диаграммы, шаблон и цветовую схему. После генерации результат можно сохранить.

### Экспорт

Сохранённую инфографику можно экспортировать в PNG. Факт экспорта фиксируется на сервере.

### Профиль

Пользователь может просматривать и редактировать личные данные, а также менять пароль.

### Администрирование

Администратору доступны:

- управление пользователями;
- изменение ролей;
- блокировка и разблокировка аккаунтов;
- просмотр аудита;
- управление шаблонами инфографики;
- резервное копирование и восстановление базы данных.

## Частые проблемы

### Клиент не подключается к серверу

Проверьте, что сервер запущен и доступен по адресу:

```text
http://localhost:8080
```

Также проверьте базовый URL API в настройках клиента.

### Ошибка file_picker

После изменения зависимостей выполните:

```bash
flutter clean
flutter pub get
flutter analyze
```

### Ошибка после обновления моделей

Если были изменены generated-файлы, выполните:

```bash
dart run build_runner build --delete-conflicting-outputs
```
