# Infographic App Server

Серверная часть веб-приложения для генерации инфографики по учебной статистике.

## Назначение

Сервер обеспечивает:

- регистрацию и авторизацию пользователей;
- выдачу JWT-токена;
- разграничение доступа по ролям;
- подключение к базе данных PostgreSQL;
- работу с учебными группами, студентами, дисциплинами и учебными периодами;
- работу с оценками и посещаемостью;
- расчет статистических показателей;
- сохранение сформированных инфографик;
- административное управление пользователями;
- просмотр журнала аудита.

## Используемые технологии

- Dart
- shelf
- shelf_router
- PostgreSQL
- pgcrypto
- JWT
- dotenv

## Настройка окружения

Создайте файл `.env` в папке `server` на основе `.env.example`.
